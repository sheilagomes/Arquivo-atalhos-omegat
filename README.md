# Arquivo-atalhos-omegat
Arquivo de configuração de atalhos no OmegaT. O arquivo vem um exemplo que altera o atalho de abertura de arquivos no OmegaT, edite-o caso não queira este atalho em especial. Mais instruções em https://omegat.sourceforge.io/manual-latest/en/appendix.shortcut.custom.html

O arquivo pode ser baixado clicando no botão verde "Clone and Download" acima, à direita, e depois na opção em "Download ZIP". Será baixado um arquivo zip (comptactado) que deve ser descompactado e depois você deve copiar o arquivo MainMenuShortcuts.properties na´pasta de configurações, acessível pelo menu Opções.
